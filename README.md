# Jiading Zhu Personal Website

Static academic personal website built with Astro and deployed to Netlify.

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```

## Deploy

Push this repository to the Git remote connected to Netlify:

```sh
git push -u origin main
```

Netlify should run `npm run build` and publish the `dist/` directory.

## Privacy Notes

The resume and LinkedIn PDFs in this working folder are source material only and are ignored by Git. They are not copied into the published site because the resume contains private contact information.
